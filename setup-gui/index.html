    Where should Astro live?

    The code installs into its own directory. Your data stays inside it under data/, snapshotted nightly.

    Fetch the code

    Astro's core repo is private. A fine-grained token with read-only Contents access on astrohome-core is all it needs — it is used for the clone and not stored.

    Create one at github.com → Settings → Developer settings → Fine-grained tokens.

    Configure the household

    Generated from the project's own .env.example — required fields are marked, secrets are pre-generated, everything can be changed later in .env.

    Astro's memory

    Start with a blank memory, or carry over everything — people, preferences, history — from a previous deployment's data.

    Reaching Astro

    Local-only works on this machine and your home network. A free Cloudflare Tunnel adds a public HTTPS address for phones and WhatsApp — no router changes.
