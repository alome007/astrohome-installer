# AstroHome installer

One-line bootstrap for [AstroHome](https://github.com/alome007) — a household AI assistant.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/alome007/astrohome-installer/main/install.sh | bash
```

You'll be prompted for your GitHub username, a PAT with `Contents: Read`
on the private `astrohome-core` repo, and (optionally) a Cloudflare
domain for public access.

Installer version: 0.1.0
